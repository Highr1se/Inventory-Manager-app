## Application Purpose
The Inventory Manager is a simple, minimalistic Flask web application designed to help small businesses track products, quantities, pricing, and low-stock alerts. The app provides a clean dashboard view, an inventory listing page, and a product creation page to manage inventory efficiently.

This project demonstrates the use of Python, Flask, HTML templates, CSS styling, and basic data persistence using JSON files.

---

## Features
- Dashboard with inventory statistics
- Add new products with SKU, price, quantity, and reorder level
- Inventory list with low-stock highlighting
- Update product quantities
- Delete products
- Clean, minimal, and responsive UI
